There are two sections here; one for MacOS and another for Windows. If you are using Linux, I assume you don’t really need these instructions, but the MacOS section will work just fine for you.

MacOS

Lucky for you, MacOS comes with a terminal. Click on your Launchpad, then type ‘terminal’. You should see an application with a console on it - you’ll want to click that.

The basic way to connect to any machine inside of Min Kao from your laptop is by typing this command into it:

ssh [netid]@[computer].eecs.utk.edu

[netid] is your netid, and [computer] is any computer inside of Min Kao that accepts ssh connections. These computers are tesla[0-30] and hydra[0-30]. For example, if I wanted to connect to hydra12, I would type:

ssh ashort11@hydra12.eecs.utk.edu

After this, it will ask you for your password. It is the same password that you use to sign into MyUTK (and everything else at UTK). When you type it, it will not show that you are typing it. You just have to know how that it is correct.

It will ask you if you trust this host, so after you type ‘yes’, you should be set to go.

There is a better way to do this that involves using a public/private key pair. Guide coming soon ™.
