CS102 - Introduction to Computer Science (Fall 2018) - Lab 1

Back to CS102 - Introduction to Computer Science (Fall 2018)

This was the altered first lab. Originally, there was slightly more detail to the lab (both cout and cin were used, but it proved very difficult to have this one added step into an already short fifty minute lab section). The main points of this lab section were to get students familiar with using a terminal, vim, Canvas. Additionally, all students should be able to connect to one of the Tesla machines from their laptop by the end of this section.

Step 1.

Log into the Tesla Machine sitting in front of you. This will be the same username/password combination that you use to log into MyUTK. Upon doing this, please open up a terminal session. You will be logged into your home directory (/home/[netid])

Step 2.

Type ls into your terminal; it should show you all the directories and files where you are. You are going to make a new directory for this class called cs102. To do this, you will type mkdir cs102. Type ls again and you should see that a new directory called cs102 has appeared. To enter this directory, type cd cs102. You are now in the path /home/[netid]/cs102.

Step 3.

You are going to make another directory for this lab, called lab1. Do this by typing mkdir lab1, and enter this directory (cd lab1). Now that you are sitting in this directory (/home/[netid]/cs102/lab1), type vim lab1.cpp, and wait until everyone catches up to this point. The programming language that will be used for this class is C++.

Step 4.

You are now in a program called Vim, a popular text editor that can be used through ssh connections. To enter Insert mode, you will hit i on your keyboard. Now, type this out into your file:

//YOUR NAME HERE
//Lab 1
//Simple Hello World program

#include <iostream>


using namespace std;

int main() {

    cout << "Hello world!\n";

    return 0;
}

Once you are done typing this, hit the esc key and type :wq. This will write the file (w), then quit out of vim (q).

Step 5.

Now that you’ve written the code, you must compile it for it do actually work. Do this by carefully typing:

g++ -std=c++11 -o lab1 lab1.cpp

The -o flag tells the compiler to put the executable to lab1. If you switch up lab1 and lab1.cpp, you will destroy all your code! Always be cautious of what you are typing.

If no errors have come up, you should be ready to run your first program! You can run your program by typing:

./lab1

After typing this, you should see Hello World! pop up on your screen. If you see Hello World!%, it’s because you forgot the \n, which prints out a newline character.

Step 6.

You have written the most basic program called “Hello World”. Submit this assignment to Canvas (the .cpp file). After submitting the file check out this to get familiar with how to log into these computers from your laptop. Below is a list of all the terminal / Vim commands used.

Terminal:

    ls          list directory contents
    cd [path]   change directory
    vim [file]  open 'Vim', the text editor you will be using in this course

Vim:

    i           enter 'Insert' mode
    esc         go to 'Normal' mode
    :w          writes the file
    :q          quits the file
    :wq         writes and quits the file
